# Pull Request

Una Pull Request (PR) és una sol·licitud de fusió d'una branca amb la branca principal. Serveix per revisar canvis abans de fusionar-los.

## Avantatges

- Permet revisió de codi.
- Facilita treball en equip.
- Manté la història de commits clara.

## Desavantatges

- Pot generar conflictes si dues persones editen el mateix fitxer.
- Si no es revisa bé, els errors poden passar a main.
