Nombre: Alan Nieto
Fecha: 03/12/2025

## Descripción

Demostrar domini de les eines de git i github.
Demostrar el dominio de las herramientas de Git y Github

## Objetivos de la tarea

- Aprender los comandos principales de Git.
- Practicar el uso de brancas y fusiones(merge).
- Mantener una estructura ordenada con los ficheros.
- Publicar los cambios en nuestro repositorio personal.
- Representar visualmente el historicos de commits y brancas con un gráfico.
